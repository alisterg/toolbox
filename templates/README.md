# Setup

Prerequisites:

- Windows, Mac or Linux
- Node v16.4.0 or greater
- Safari v15.5

Installation:

1. Run `npm install`
2. `npm run dev` to host at localhost:3000
3. `npm run test:unit` to run the test suite

Deployment:

1. `npm run build` to build to dist/
2. `serverless deploy` to deploy to the cloud
